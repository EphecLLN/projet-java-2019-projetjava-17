INSTALLER ET UTILISER

1. tous les fichiers .java sont dans /src -> les compiler
2. le main se trouve dans Quizz.java
3. le fichier regles.txt doit se trouver le dossier d'exécution du programme
4. il faut la lib java /lib/mysql-connector-java-5.0.5-bin.jar
5. il faut une base de données mysql qui tourne, qui peut être remplie à l'aide du fichier SQL /db/DBprojetJava.sql
	La config sql est hardcodée dans le fichier Partie.java:
	
	public static String url = "jdbc:mysql://localhost/projetjava";
	public static String login = "root";
	public static String passwd = "";
	
6. le jeu se lance en console part défault, pour le lancer en gui, il faut que le premier paramètre de la ligne de commande soit "gui"
